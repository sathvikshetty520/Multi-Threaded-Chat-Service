# Multithreaded Chat Service using Java

## Project Overview

This project is a simple Multithreaded Chat Service developed using Java Socket Programming and Multithreading concepts.

The application allows multiple clients to connect to a server and communicate simultaneously through a shared chat system.

Each client is handled using a separate thread, enabling real-time communication between multiple users.

---

# Features

- Multiple client connections
- Real-time message broadcasting
- Multithreading support
- Client disconnect handling
- Socket programming implementation
- Simple and easy-to-understand code

---

# Technologies Used

- Java
- Socket Programming
- Multithreading
- VS Code / Command Prompt
